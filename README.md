# نظام إدارة الطلاب

## وصف المشروع

برنامج Console بسيط بلغة C# لتطبيق أساسيات البرمجة الكائنية (OOP) واستخدام القوائم والدوال والتحقق من صحة المدخلات.
## أهداف المشروع

- تطبيق أساسيات لغة C#.
- استخدام الكلاسات (Classes) والكائنات (Objects).
- التعامل مع القوائم (List).
- استخدام الشروط والحلقات والدوال.
- التحقق من صحة بيانات المستخدم.

## وظائف البرنامج

- إضافة طالب جديد.
- عرض جميع الطلاب.
- البحث عن طالب باستخدام الرقم.
- تعديل بيانات الطالب.
- حذف طالب.
- إدخال علامات لكل طالب.
- حساب متوسط علامات كل طالب.

## الملفات الموجودة في المشروع

- Program.cs
- Student.cs
- README.md

## طريقة تشغيل البرنامج

1. فتح المشروع باستخدام Visual Studio.
2. تشغيل المشروع.
3. اختيار العملية المطلوبة من القائمة الرئيسية.

## التحقق من صحة المدخلات

- التأكد من أن رقم الطالب رقم صحيح.
- منع تكرار رقم الطالب.
- التأكد من أن اسم الطالب غير فارغ.
- منع إدخال أرقام داخل اسم الطالب.
- التأكد من أن العلامات تقع بين 0 و100.
